FAQ

What/Why/When/Who/Where is totallynuclear.club?

Totallynuclear.club is a simple, tiny linux machine run for social purposes inspired by Paul Ford’s tilde.club. (You can read more about how all of that happened here.)

As a sysadmin of a tiny linux server back in college (mrfalcon.net) and a member of a less tiny and way more awesome linux server before that (berzerker.net), Ford’s Medium article made me nostalgic for those days and reminded me how valuable those experiences were for me (if for nothing else, to learn my way around the unix shell). A few beers later, tada!

I launched totallynuclear.club right around 2014-10-11T15:45:00-05:00

My name is Erik Price. I can be found on totallynuclear.club at totallynuclear.club/~erik

totallynuclear.club resides on an Amazon EC2 instance. I’m told somewhere in Virginia.

What is totallynuclear.club not?

A robust hosting platform where you should host critical content.

Where does the name “totally nuclear” come from?

That was the embarrassing reply I gave to a girl in 7th grade who told me that another girl had the hots for me. (I probably stole it from Cody on Step by Step, but that’s never been confirmed.)

What can I expect from totallynuclear.club?

I am copying and pasting this straight from Paul Ford’s Sacred Vows in his Medium article and pledge to do the same:

  1. I will make a weekly backup of the public_html directories so when some teen in Estonia decides to hack in and ruin everything we can bring up a new server and limp back to life.
  2. I won’t shut things down without a month of warning and once it’s shut down I’ll make sure a file with all the public_html directories is uploaded to archive.org.
  3. If any community forms at all (DUBIOUS BUT SURPRISE ME) I promise I won’t blow up the community without, like, first pointing everyone to some free IRC channel or something.

Beyond that, you should know that this is a very informal project and subject to all sorts of uptime issues, unreliability of service and unreliability of attention from me.

The server is down! What do I do?

Wait. I almost certainly already know about it and am working on it. If you feel you must, you can hit me up on twitter at @erikprice (just once is good, thanks).

Can you install my favorite software?

No.

What if I have a problem or something is busted?

Check out the Github repo and see if the issue already exists. If not, create it.

Are there any rules?

Don’t be a jerk. Jerks will be unceremoniously banned. You agreed not to be a jerk, remember?

What can I do with this thing?

  1. Check out some recently updated pages for inspiration.
  2. See a list of users who are on the server with command ‘users’
  3. Message any totallynuclear.club member; type in ‘mutt’ to access the mail client or ‘talk whatever_username’ to chat directly.

How do I log in?

Open up terminal on your computer. Type in ssh your_username@totallynuclear.club and your temporary password (which should have been emailed to you). You should change your password right away. Type in passwd, hit enter, and follow the prompts. Set up public key authentication, which will soon be required. Here are directions for enabling public key authentication.
